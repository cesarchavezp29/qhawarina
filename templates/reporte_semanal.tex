  \small\color{ink}
  \textbf{Índice de Precios BPP}\\
  Metodología Jevons bilateral con datos de supermercados (Plaza Vea, Metro, Wong).
  Filtro de ratio extremo: $0.5 < r < 2.0$. Frecuencia: diaria. Base = 100 al inicio del período.\\[6pt]

  \textbf{Nowcast PBI}\\
  Modelo de Factores Dinámicos (DFM) con $k=3$ factores, 33--37 series del panel nacional.
  Bridge equation con regularización Ridge ($\alpha = 1.0$). Ventana rodante de 7 años.\\[6pt]

  \textbf{Índice de Riesgo Político}\\
  Clasificación NLP sobre 6+ fuentes RSS. Score normalizado 0--1 (componente financiero +
  componente textual). Índice de inestabilidad con ajuste estacional.\\[6pt]

  \textbf{Nowcast Pobreza}\\
  Gradient Boosting Regressor con variables rezagadas de PBI, inflación y serie NTL (NASA VIIRS).
  Panel departamental 2004--2024. RMSE: $\sim$2.5pp; Rel.RMSE vs AR1: 0.95.\\[6pt]

  \textbf{Mercado Cambiario}\\
  Datos de BCRP API (PD04638PD). Intervenciones spot (PD04659MD) y swaps (PD04660MD).
  Tasa de referencia BCRP (PD12301MD).
\end{tcolorbox}
